# Kubernetes Projects Portfolio

This repository contains a collection of practical Kubernetes projects designed to showcase hands-on experience with container orchestration, deployment, and cluster management.  

## Overview

These projects cover essential Kubernetes concepts and best practices, including:

- **Pods, Deployments, and Services**  
- **Scaling and High Availability**  
- **Persistent Volumes (PV) and Persistent Volume Claims (PVC)**  
- **Taints and Tolerations**  
- **Containerization and Docker image management**  
- **Versioning and rolling updates**

Each project is organized in its own folder with application code, Dockerfiles, and Kubernetes manifests, making it easy to explore and deploy in a real Kubernetes environment.

---

## How to Use

1. Navigate to the project folder of interest.  
2. Build the Docker image if required.  
3. Apply the Kubernetes manifests using `kubectl apply -f <folder>/k8s/`.  

> This repository is meant as a practical portfolio to demonstrate Kubernetes and DevOps skills.
